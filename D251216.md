# Flutter 에서 진짜 자주 사용하는 위젯!

- 위젯: 화면에 보이는 모든 요소

1. [***MaterialApp***](https://api.flutter.dev/flutter/material/MaterialApp-class.html)
- 모든 앱에서 반드시 한번은 사용되는 위젯
- 머티리얼 디자인을 사용하는 애플리케이션
- 머티리얼 디자인: 구글에서 디자인한 가이드. 이름을 머티리얼이라고 지음
- 페이지 설정(앱 내에서 사용되는 페이지들)
- ⭐️ **Navigator 스택 관리**
- **테마 관리**
- 다국어 설정

2. [***Scaffold***](https://api.flutter.dev/flutter/material/Scaffold-class.html)
- 페이지 골격을 잡아주는 위젯
- 페이지별로 하나씩!
- 많이 나뉘어진 레이아웃을 미리 구글에서 나눠놓은 위젯
- appBar 영역: 상단. 보통 AppBar라는 위젯이 들어감!
- body 영역: 가운데 컨텐츠
- bottomNavigatorBar, bottomSheet 영역: 하단
- 앱을 만들면 MaterialApp 이 들어간 위젯클래스 만들기
- 헤이지를 만들 땐 Scaffold가 들어간 위젯클래스 만들기
- 하나의 클래스는 하나의 파일에서 관리 권장
- 반드시 appBar 영역과 bottom 영역을 사용해야하는건 아님
- 특수한 경우 body 안에서 구현하는게 더 편함! (by 박진251216)

3. [***AppAbr***](https://api.flutter.dev/flutter/material/AppBar-class.html)
![AppBar](https://flutter.github.io/assets-for-api-docs/assets/material/app_bar.png)

- Scaffold의 상단영역에 배치할 수 있는 위젯
- 상단영역을 leading, title, actions 등의 영역으로 미리 골격을 나눠놓은 위젯!
- leading: 뒤로가기 버튼 주로 사용, 네베게이션 스택을 보고 이전페이지가 존재하면 자동으로 뒤로가기 버튼을 만들어줘서 안써줘도 됨, 단 뒤로가기 아이콘을 바꾸로 싶을 땐 구현!
- **title**: 제목. 가운데 글자. 안드로이드에서는 디폴트로 왼쪽.
- centerTitle: true로 주면 안드로이드에서도 제목이 가운데 정렬!
- **actions**: 위젯이 들어가는 리스트. 리스트 안에 들어가 있는 위젯들이 차례대로 배치됨!
